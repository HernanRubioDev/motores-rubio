Motores-Rubio es una aplicacion de tipo REST para administrar talleres de bobinado de motores eléctricos.
La idea de crear este programa, fue para aprender sobre los fundamentos REST, asincronismo, validaciones tanto de formularios como a la hora de hacer peticiones HTTP, manejo de rutas y Bootstrap.

Las tecnologias utilizadas fueron HTML, CSS, Bootstrap, Javascript y ReactJs. En cuanto a la persistencia de datos, utilicé Json-Server.
